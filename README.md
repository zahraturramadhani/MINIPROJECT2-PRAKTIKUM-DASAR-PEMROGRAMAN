# Menu User

# Penjelasan Menu-Menu User

1. Lihat Data Diri Sendiri
   
   Untuk masuk ke Menu "Lihat Data Diri Sendiri" User harus menginputkan angka 1
   
   ![Cuplikan layar 2024-10-14 195849](https://github.com/user-attachments/assets/dfe80003-0d70-46c8-86fc-b14798823af8)

   Untuk melihat data Anda, Anda cukup menginputkan id karyawan Anda maka akan muncul semua data milik Anda dan Setelah itu juga User akan ditanya kembali apakah 
   ingin melihat data Anda lagi dengan sistem looping (ya/tidak)
   
   ![Cuplikan layar 2024-10-14 200548](https://github.com/user-attachments/assets/ad60cbee-5ce5-410c-88cf-a3663a9366e0)

    * Jika User menginputkan "ya" otomatis User akan terlooping ke input "Masukkan Id Anda " seperti pada contoh diatas.

      ![Cuplikan layar 2024-10-14 201034](https://github.com/user-attachments/assets/7e803586-56c7-42b2-89ee-31963bec6d80)

    * Jika Admin menginputkan "tidak" otomatis Admin akan kembali ke Menu User.

      ![Cuplikan layar 2024-10-14 201139](https://github.com/user-attachments/assets/528501eb-0232-4b14-ab59-cbf97793f09b)

2. Ambil Cuti Karyawan
   
   Untuk masuk ke Menu "Ambil Cuti Karyawan" User harus menginputkan angka 2

   ![Cuplikan layar 2024-10-14 201430](https://github.com/user-attachments/assets/34274902-639e-4959-a64a-073a05f8d7b0)

   Untuk mengambil cuti, Anda harus menginputkan Nama dan Id karyawan Anda serta jumlah cuti yang ingin diambil maka akan muncul jumlah cuti yang anda ambil dan 
   juga sisa cutinya
   
   ![Cuplikan layar 2024-10-14 201828](https://github.com/user-attachments/assets/32a71d45-afd0-4e9e-ad11-761214cb0411)

   * Jika jumlah cuti yang diambil <= sisa cuti, maka jumlah cuti yang diinginkan akan bisa terambil, outputnya bisa dilihat seperti gambar dibawah ini.

     ![Cuplikan layar 2024-10-14 203539](https://github.com/user-attachments/assets/0bd9eb80-cfd6-4c69-90f9-749ad19ef4dc)

  * Jika jumlah cuti yang diambil >= sisa cuti, maka jumlah cuti yang diinginkan tidak bisa terambil, ouputnya bisa dilihat seperti gambar dibawah ini.

    ![Cuplikan layar 2024-10-14 202552](https://github.com/user-attachments/assets/1d61a3c4-2b97-4844-8d4b-369b2119904f)

  Setelah itu juga User akan ditanya kembali apakah ingin mengambil cuti lagi dengan sistem looping (ya/tidak)
  
   * Jika User menginputkan "ya" otomatis User akan terlooping ke input "Masukkan Nama Anda " sampai input "jumlah cuti yang ingin diambil" seperti pada contoh 
     diatas.
     
     ![Cuplikan layar 2024-10-14 202952](https://github.com/user-attachments/assets/6a450998-c11d-4307-ae15-5d92669dfd9e)

   * Jika Admin menginputkan "tidak" otomatis Admin akan kembali ke Menu User.

     ![Cuplikan layar 2024-10-14 203254](https://github.com/user-attachments/assets/6babe163-cb2b-41e5-a26d-eea89e920f00)
